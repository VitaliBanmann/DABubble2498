# DA Bubble

DA Bubble is the group project for a team chat application. This repository contains the Angular app, Firebase configuration, and supporting assets.

## Project structure
- src/: Angular application source
- assets/: Images and static files
- environments/: Environment config

## Quick start
1. Install dependencies
	- npm install
2. Run the app
	- npm start

## Build
- npm run build

## Notes
- This repository is only for the DA Bubble project.
- Firebase configuration lives inside this project.
